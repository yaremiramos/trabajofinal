Listas Ordenadas

Que son las Listas Ordenadas
Son aquellas en las que cada elemento se encuentra precedido por un número, o letra que representa la posición que dicho elemento ocupa en la lista. Las lista ordenadas se definen con las etiquetas y cada elemento de la lista. Aunque no es necesario, muchos programas generadores de páginas HTML utilizan la etiqueta LI como si fuese cerrada.
Ejemplo de Listas Ordenadas

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
  4. Cuarto elemento
